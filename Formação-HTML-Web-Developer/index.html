Estruturas Básica do HTML

O HTML (Linguagem de Marcação de Hipertexto) é a estrutura fundamental por trás de qualquer site. Ele é composto por "tags", que são instruções específicas que dizem ao navegador como exibir conteúdo. Cada tag tem um propósito definido, Essas tags podem ser aninhadas, ou seja, colocadas umas dentro das outras, para criar estruturas mais complexas. Além disso, o HTML interage com outras linguagens, como CSS e JavaScript, para criar páginas dinâmicas e estilizadas. Em resumo, o HTML é a espinha dorsal de qualquer página da web, fornecendo sua estrutura básica e conteúdo.

Falando Sobre Tags

Tags são elementos fundamentais em linguagens de marcação, como o HTML. Elas atuam como marcadores que definem como diferentes partes do conteúdo são interpretadas e exibidas. Cada tag é cercada por sinais de menor e maior, como . Muitas delas trabalham em pares uma tag de abertura, como , e uma de fechamento, como . Estas delimitam e especificam o tipo e a função do conteúdo entre elas, seja para formatar texto, inserir imagens, criar links ou estruturar a página. A correta utilização das tags é essencial para a organização e apresentação adequada dos sites na web.

Atributos das Tags

Atributos em tags HTML especificam características adicionais ou comportamentos dos elementos. O atributo style permite adicionar estilos CSS diretamente, como style="color: red;". Atributos relacionados ao JavaScript, como onclick, definem ações a serem tomadas ao ocorrer um evento. O elemento input frequentemente usa atributos como type, que determina seu formato (por exemplo, "text" ou "password"). O atributo href é específico para a tag , indicando o URL de destino do link. Juntos, esses atributos enriquecem a funcionalidade e apresentação das páginas web.

Listas Ordenadas e Não Ordenadas

No HTML, existem listas ordenadas e não ordenadas. As listas ordenadas, usando a tag <'ol'>, apresentam itens em sequência numerada. Já as listas não ordenadas, com a tag <'ul'>, exibem itens com marcadores, sem uma ordem específica.

    Para fazer o bolo siga os passos

  1. Pré-aqueça o forno a 180°C e unte uma forma de bolo com manteiga ou óleo
  2. Em uma tigela grande, peneire a farinha de trigo, o açúcar, o fermento e o sal. Misture bem.
  3. Em outra tigela, bata os ovos, acrescente o leite, o óleo ou manteiga derretida e a essência de baunilha. Misture até obter uma massa homogênea.
  4. Despeje lentamente os ingredientes úmidos sobre os secos, mexendo continuamente até que a massa esteja lisa e sem grumos.
  5. Despeje a massa na forma preparada e leve ao forno por cerca de 25-30 minutos ou até que, ao inserir um palito no centro, ele saia limpo.
  6. Após retirar do forno, deixe o bolo esfriar na forma por alguns minutos. Em seguida, desenforme e, se desejar, decore com cobertura ou polvilhe com açúcar de confeiteiro.

Falando um pouco sobre Links

Links, também conhecidos como hiperlinks, são elementos fundamentais da web. Eles permitem a navegação de uma página para outra, seja dentro do mesmo site ou para um site externo. Em HTML, os links são criados usando a tag <'a'>.

Alguns exmplos sobre a tag <'a'>

  • <'a'>: A tag que indica o início de um link.
  • href: Atributo que especifica o URL (Uniform Resource Locator) para o qual o link aponta. Pode ser um caminho para um arquivo local, um endereço de página web, ou até um endereço de email.
  • Texto do Link: O texto visível que os usuários clicam. Pode ser substituído por uma imagem ou outro elemento HTML.
  • <' /a'>: A tag que indica o final do link
